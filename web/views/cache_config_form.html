Configuração da Cache
bits

Aqui você determina quantos bits são necessários para representar o endereço da memória física. Você pode colocar entre 16 bits e 128 bits para o endereço da memória física.

bits

Aqui você determina quantos bits são necessários para representar o offset de cada bloco da cache. Você só pode colocar entre 8 (byte) e 128 bits (quadword).

bits

Aqui você determina quantos bits são necessários para representar a faixa de índices (linhas da cache, entradas da cache, etc). Você só pode colocar entre 0 e 128 como tamanho do campo de índice.

bits

Aqui você determina quantos bits são necessários para representar a tag. No mínimo 8 bits e no máximo 120. Pois nessa aplicação você tem um mínimo de 8 bits de offset. Existem outros tipos de cache com tamanhos diferentes.
